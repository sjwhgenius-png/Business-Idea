## Deploy on Netlify

Use these settings:

Base directory:

```txt
ampra-electrical-premium
```

Build command:

```txt
npm run build
```

Publish directory:

```txt
dist
```

## Local preview

```bash
cd ampra-electrical-premium
npm install
npm run dev
```
